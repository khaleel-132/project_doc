In this project we will create an online tutor finding system in which students can find the respective tutors for their subjects and tutors can find respective students for private tutions.


Django Installation

Firstly You nee to install the django and other requirements for starting this project.for this go to start menu of your system and search for CMD(Command Prompt).After selecting CMD create a root folder for our project

mkdir TutorProject

Before installing django make sure that python is installed on your Pc.To check wheather python exists...

python --version

If not installeds install python using

pip install python

Then You need a virtual environment for installing and creating django applications.for this type

virtual tutorenv

This will create a virtual evironment named tutorenv in your system

After creating go to files like scripts and activate to activate virtual environment for this

tutorenv\Scripts\activate

Now we need to install the latest version of django

pip install django

After installtion we need to create a django project folder which will include all the dependencies for running django

django-admin startproject Tutorproject

Here a new project is created...and goto that project folder using

cd Tutorproject

In this project we need to create a new app to develope our application.For this

python manage.py startapp tutorapp

this will create a neww app on your project.

So you are done with the installation and now we have to run our django project locally for this type command...

python manage.py runserver

This will result like this.

Project Setup

Open TutorProject Folder with any IDE.

Your created App will look like this

tutorapp/
__init__.py
admin.py
apps.py
migrations/
__init__.py
models.py
tests.py
views.py

in the tutorapp create a file urls.py and include following code

tutorapp/urls.py
from django.urls import path
from . import views
urlpatterns = [
path('', views.index, name='index'),
]

The next step is to point the root URLconf at the tutorapp.urls module. In Tutorproject/urls.py, add an import for django.urls.include and insert an include() in the urlpatterns list, so you have:

Tutorproject/urls.py
from django.contrib import admin
from django.urls import include, path
urlpatterns = [
path('tutorapp/', include('polls.urls')),
path('admin/', admin.site.urls),
]

In settings.py of the Tutorproject include in INSTALLED APPS like this.

Tutorproject/sttings.py
INSTALLED_APPS = [
'django.contrib.admin',
'django.contrib.auth',
'django.contrib.contenttypes',
'django.contrib.sessions',
'django.contrib.messages',
'django.contrib.staticfiles',
'tutorapp'
]
